# PayMeBack

App om voorgeschoten bedragen tussen mensen bij te houden en te verrekenen.
React + TypeScript (Vite), Tailwind CSS, Supabase. Leerproject voor React.

## Setup

1. `npm install`
2. Kopieer `.env.example` naar `.env` en vul je Supabase-gegevens in.
3. `npm run dev` en open http://localhost:5173

## Scripts

- `npm run dev` — dev-server
- `npm run build` — productiebuild
- `npm run format` — code formatteren met Prettier
